Установка за минуту

Одна команда запускает мастер, который настроит всё за тебя.

  1. 1

    Запусти установщик

    Мастер care-install склонирует репозитории, создаст .env и поднимет сервисы.

    $ uvx care-install

    Нет uv? Как установить uv

    $ curl -LsSf https://astral.sh/uv/install.sh | sh
    $ pipx run care-install

    Нет pipx? Как установить pipx

    $ python3 -m pip install --user pipx
  2. 2

    Запусти MAESTRO CARE

    Мастер добавит команду care в PATH — открой чат из любой директории.

    $ care

Что нужно

  • uv (для uvx) или pipx
  • git в PATH
  • Доступ к LLM с OpenAI-совместимым API
  • Запущенный Docker — для локального запуска навыков в песочнице (без него — простой режим скачивания) скачать Docker

Рекомендуемые модели

Qwen3-Coder · GLM-4.6 · Kimi 2.6

Выбирай модель с уверенным tool-calling и контекстом от 32K — цепочки собираются стабильнее и выше по качеству.

Обновить до последней версии:
$ uvx --refresh care-install

Агентный навык

Добавь CARE как навык своему агенту

Дай агенту управлять CARE CLI — Claude Code, Codex, Hermes или OpenClaw.

  1. 1

    Скачай навык

    care-cli.skill — это бандл, который учит агента находить и запускать care.

    $ curl -LO https://airi-maestro.github.io/care-docs/care-cli.skill
  2. 2

    Распакуй в свой агент

    Выбери агента — навык встанет в его каталог skills.

    $ unzip care-cli.skill -d ~/.claude/skills/
    $ unzip care-cli.skill -d ~/.codex/skills/
    $ unzip care-cli.skill -d ~/.hermes/skills/
    $ unzip care-cli.skill -d ~/.openclaw/skills/

После установки

  • Нужна рабочая команда care (шаги выше)
  • Вызови /care-cli или опиши задачу словами
Проверь навык:
$ care doctor
