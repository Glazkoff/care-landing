Анимация интерфейса

Установка за минуту

Три команды — и CARE запущен. Без сборки.

  1. 1

    Установи зависимости

    uv создаст окружение и поставит весь стек CARE.

    $ uv sync
  2. 2

    Укажи доступ к модели

    care init создаст файл .env: адрес сервиса, ключ и название модели.

    $ uv run care init
  3. 3

    Запусти CARE

    Откроется чат — опиши задачу и собери первого агента.

    $ uv run care

Что нужно

  • Python 3.12+ и uv
  • Доступ к LLM с OpenAI-совместимым API
  • Хранение и эволюция — опционально, для режима Production
Одной строкой:
$ uv sync && uv run care init && uv run care
