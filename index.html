Установка за минуту

Три команды — и CARE запущен. Без шага сборки.

  1. 1

    Синхронизируй зависимости

    uv создаст окружение и поставит весь стек CARE.

    $ uv sync
  2. 2

    Запиши доступы к MAGE

    care init создаст минимальный .env: base URL, API-ключ и модель.

    $ uv run care init
  3. 3

    Запусти TUI

    Откроется chat-поверхность — опиши задачу и собери первого агента.

    $ uv run care

Что нужно

  • Python 3.12+ и uv
  • Доступ к MAGE-совместимой LLM
  • Memory и Platform — опционально, для режима Production
Одной строкой:
$ uv sync && uv run care init && uv run care
